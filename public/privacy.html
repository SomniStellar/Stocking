Stocking Privacy Policy

Last updated: 2026-04-24

Stocking is a personal portfolio tracking web application that uses Google Sign-In and Google Sheets chosen by the user.

What data Stocking accesses

  • Basic Google account profile information needed for sign-in
  • Google Sheets data from spreadsheets the user explicitly connects or creates

How data is used

  • To authenticate the user
  • To read, create, and update the spreadsheet data required by the app
  • To display the user portfolio, watchlist, and benchmark information inside the app

Data storage

Stocking operates as a browser-based application. Portfolio data is primarily stored in the user's own Google Sheets. Authentication tokens are used only to perform the actions the user requests in the current session.

Data sharing

Stocking does not sell user data. Data is only sent to Google APIs that are required for sign-in and spreadsheet operations initiated by the user.

User control

Users control their spreadsheet access through their Google account and spreadsheet sharing settings. Users can stop using the app at any time and can revoke Google access from their Google account permissions page.

Contact

For support or privacy-related questions, visit the support page.
