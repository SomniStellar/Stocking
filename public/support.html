Stocking Support

Last updated: 2026-04-24

Stocking is a Google Sheets-based portfolio tracking application for monitoring holdings, benchmarks, and watch items.

What users need

  • A Google account
  • Permission to access or create a Google Sheet for portfolio data
  • A modern desktop or mobile browser

Supported flow

  • Sign in with Google
  • Connect an existing spreadsheet or create the template from the app
  • Manage holdings and dashboard data from the connected sheet

Common issues

  • If Google Sign-In is unavailable, verify the app owner configured the OAuth client ID.
  • If sheet access fails, make sure the connected account can access the target spreadsheet.
  • If the app shows an unverified warning, the owner has not completed Google OAuth verification yet.

Policy

Review the privacy policy for details on data access and usage.
