# Dashboard Chart Status

## Current State
- The dashboard range chart is implemented with `Recharts`.
- Period toggle is `YTD`, `3Y`, `5Y`.
- The toggle switches the whole chart range, not simultaneous period points.
- Benchmark cards are rendered before the chart and remain the primary comparison UI.
- On live load, `Monitor` is synchronized first and `Series` is refreshed in the background.
- Because of that, the chart can briefly stay in a pending state on first load.

## Current Rules
- Custom benchmark cards keep only inline delete.
- Benchmark enable or disable is triggered by clicking the card.
- Failed benchmarks stay visible as cards but are excluded from chart comparison.
- Chart width follows the same centered fixed-width rule as the benchmark card grid.

## Related Holdings Rule
- Holdings input is treated as current position adjustment.
- It is not a transaction-history editor.
- The side control uses short labels `Add` and `Reduce` so the control stays inside the fixed card width.

- Preserve chart shell height if lazy mount is introduced
