# 문서 마스터맵

## 시작 기준
- 사용자 검토는 이 문서에서 시작한다.
- 현재 기준 문서는 이 문서에 적힌 경로만 사용한다.
- `.codex` 문서는 사용자 검토 기준 문서가 아니다.

## 운영 원칙
- 기능 구조가 바뀌면 본편 3종과 이 마스터맵을 같은 턴에 같이 갱신한다.
- 결정 완료 내용은 `결정사항` 문서로 이동하고, 더 논의가 필요한 항목만 `미결정사항` 문서에 남긴다.
- `.codex`는 Codex 참조용이고, `.docs`는 사용자 검토용이다.
- 경로가 바뀌면 이 문서를 먼저 갱신한다.
