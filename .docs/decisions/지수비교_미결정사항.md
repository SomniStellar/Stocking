# 지수비교 미결정사항

## 목적
- 지수비교와 구글시트 작성에서 아직 사용자 판단이 필요한 항목만 남긴다.
- 이미 확정된 범위와 기술 선택은 이 문서에 다시 적지 않는다.

## 현재 우선 결정 필요
1. `KOSPI`, `KOSDAQ` fallback 후보 티커 확정
- 직접 지수는 primary로 사용한다.
- fallback은 인덱스 펀드 또는 ETF 후보로 둔다.
- 적절한 후보를 찾지 못하면 빈 값으로 유지한다.

## 구현 직전 확정 필요
1. fallback 사용 표시 문구 기준
2. 오류 캡션 문구 기준
3. 디버그 export 플래그 이름 및 위치
4. 포트폴리오 계산 결과 검증 기준

## 후속 검토 조건
- 차트 1차 범위를 넘는 상호작용이 필요하면 차트 UX와 라이브러리 범위를 다시 검토한다.
